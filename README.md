# Correlation Heatmap using Python

## 📌 Project Overview
This project is part of my Data Science Internship tasks.

The objective of this project is to visualize relationships between numerical variables using a correlation heatmap.

## 📂 Dataset

## 🛠 Tools & Libraries Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Google Colab

## 📊 Steps Performed

4. Generated correlation matrix
5. Created heatmap visualization using Seaborn

The heatmap shows correlations between different numerical features in the dataset.
